# ООО «Обувь» — практическая работа №1

## Что реализовано

- модуль 1: проектирование БД, SQL-скрипт, импорт исходных данных;
- модуль 2: авторизация, роли, просмотр товаров, подключение БД, логотип и иконка;
- модуль 3: последовательный интерфейс, поиск, сортировка, фильтрация, добавление/редактирование/удаление товаров, обработка ошибок;
- модуль 4: просмотр, добавление, редактирование и удаление заказов;
- линтер: настройка Ruff через `pyproject.toml`;
- рефакторинг: логика разделена на слои `repository`, `services`, `ui`, `models`.

## Структура проекта

- `main.py` — точка входа;
- `app/` — исходный код приложения;
- `database/init.sql` — скрипт создания БД и импорта данных;
- `database/shoe_store.db` — локальная SQLite БД;
- `docs/er_diagram.pdf` — ER-диаграмма;
- `docs/algorithm_flowchart.pdf` — блок-схема алгоритма;
- `docs/screenshots.docx` — документ со скриншотами;
- `resources/images/` — изображения товаров, логотип и заглушка;
- `scripts/reset_db.py` — пересоздание базы данных.
